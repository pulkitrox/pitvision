— THE PROJECT

A dissertation-grade
strategy engine.

PitVision AI predicts whether a Formula 1 driver should pit on the next lap using a machine learning stacking ensemble. It learns from years of lap-level race telemetry to detect the precise moment when tyre degradation, lap-time delta and field position make a pit stop the optimal call.

— OBJECTIVES

Project objectives

  1. 01

    Quantify pit decisions

    Convert the qualitative intuition of race engineers into a calibrated binary classification problem.

  2. 02

    Handle severe class imbalance

    Roughly 3% of laps end in a pit stop. The model is optimised for recall to avoid missing pit-windows.

  3. 03

    Be explainable

    Feature importance & meta-learner coefficients make the recommendation auditable for engineers.

  4. 04

    Be deployment ready

    Designed to plug into a Flask backend serving JSON predictions for live strategy software.

— METHODOLOGY

How it works

Feature Engineering

Forty-three engineered features capture lap-time evolution, rolling pace statistics, position trajectory, degradation slope and stint progress for every lap.

Ensemble Learning

Three independent base learners — Random Forest, XGBoost and LightGBM — are trained on out-of-fold splits and combined via stacking.

Hyperparameter Tuning

Each base learner is tuned with Optuna over 100 trials, optimising PR-AUC on the 2024 validation season.

Logistic Regression Meta-Learner

A logistic regression meta-classifier learns the optimal blend of base learner probabilities for the final pit decision.

— STACK

Algorithms

Random Forest

Robust baseline ensemble of decision trees with bagging. Captures non-linear interactions between stint length and degradation.

XGBoost

Gradient boosted trees with regularisation. Excels at handling the imbalanced positive class.

LightGBM

Histogram-based gradient boosting. Fast on the wide feature matrix and a strong recall driver.

Logistic Regression

Calibrated meta-learner producing the final pit probability from base-learner outputs.
